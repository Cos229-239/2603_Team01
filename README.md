# 2603 Team01

Class project — mobile app built with **Expo** and **React Native**.  
This repo also contains course documentation and planning materials.

### App location

- **Expo app root**: `2603_Team01_Expo`
- The old bare React Native app under `2603_Team01` has been retired in favor of Expo.

### How to run the app (Expo Go)

From the project root directory `DevReflect`:

```sh
cd 2603_Team01_Expo
npx expo start -c
```

Then:
- Install **Expo Go** on your device.
- Scan the QR code shown in the terminal or browser to open the app.

### Tech stack

- **Expo SDK 54**
- **React Native 0.81**
- **React 19**

### Notes for teammates

- Make all future mobile app changes inside `2603_Team01_Expo` (for example, edit `App.tsx` there).
- You do **not** need Android Studio or Xcode just to run the app in Expo Go.
